Pokémon Shiny Value (TSV/ESV) Checker


This tool will let you check the "shiny value" of a player or of an egg (a value between 0 and 8192 for Gen III to V, and between 0 and 4096 for Gen VI and up). If the egg and the player's SVs are the same, the resulting Pokémon will be hatched shiny. This tool can also calculate Pokémon Shiny Value (PSV), just use the ESV section.

Note: This tool is only accurate for Pokémon games Gen III and up. Don't input decimal PIDs—convert them to hexadecimal first.
